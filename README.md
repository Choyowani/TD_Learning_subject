# TD Learning in Random-Wall GridWorld

## 프로젝트 개요
이 프로젝트는 4x4 GridWorld 환경에서 에이전트가 **랜덤 정책**으로 이동할 때,  
**TD(0) Learning**을 이용해 상태가치 \(V(s)\)를 학습하는 예제입니다.

다음 두 환경을 비교합니다.

- **일반 GridWorld**: 벽이 없는 4x4 격자
- **랜덤 벽 GridWorld**: 매 에피소드 시작 시 벽이 2개 또는 3개 랜덤 생성되는 4x4 격자

---

## 환경 설정

- 격자 크기: 4x4
- 시작 위치: `(0, 0)`
- 목표 위치: `(3, 3)`
- 행동:
  - `0`: 왼쪽
  - `1`: 위
  - `2`: 오른쪽
  - `3`: 아래
- 보상:
  - 일반 이동: `-1`
- 종료 조건:
  - 목표 위치 `(3, 3)` 도착 시 에피소드 종료

### 벽 생성 조건
- 매 에피소드 시작 시 벽 랜덤 생성
- 벽 개수: `2개 또는 3개`
- 시작 위치와 목표 위치에는 벽 생성 불가
- 같은 위치에 중복 생성 불가
- 벽이 있는 칸으로는 이동 불가

### 이동 규칙
- 격자 밖으로 이동하려 하면 제자리 유지
- 벽이 있는 칸으로 이동하려 하면 제자리 유지

---

## 학습 방법

이 프로젝트는 **TD(0) Learning**을 사용합니다.

### 업데이트 식
\[
V(s) \leftarrow V(s) + \alpha \left[r + \gamma V(s') - V(s)\right]
\]

- \(V(s)\): 현재 상태의 가치
- \(r\): 보상
- \(s'\): 다음 상태
- \(\alpha\): 학습률
- \(\gamma\): 할인율

에이전트는 매 step마다 행동을 **랜덤하게 선택**합니다.
